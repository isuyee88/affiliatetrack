# AffiliateTrack 开发团队架构

## 项目概述
对标 Keitaro 的 Affiliate 流量跟踪管理系统
技术栈: Cloudflare Workers + D1 + Durable Objects
部署平台: Cloudflare 免费账户

---

### 核心功能模块

1. **流量跟踪模块**
   - 点击跟踪 (Click Tracking)
   - 展示跟踪 (Impression Tracking)
   - 转化跟踪 (Conversion Tracking)
   - Postback 处理

2. **流量分发模块**
   - 智能分发规则
   - A/B 测试
   - 权重分发
   - 地理分发
   - 设备分发

3. **报表分析模块**
   - 实时报表
   - 历史报表
   - 自定义报表
   - 导出功能

4. **用户管理模块**
   - 用户认证
   - 角色权限
   - API 密钥管理

5. **集成模块**
   - CPA 网络集成
   - 第三方 API
   - Webhook 通知

每个阶段结束时进行质量门禁检查:

```yaml

    criteria:

      - 性能目标明确

    reviewer: Developer Agents
```
